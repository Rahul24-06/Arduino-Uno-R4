## Project Overview

This project demonstrates:
- How to use the **Arduino Uno R4 WiFi onboard LED matrix**
- How to render **custom bitmap graphics**
- Clean usage of the official Arduino LED Matrix library

It is ideal for:
- Beginners learning matrix displays
- Hackathon / contest submissions
- Valentine, pop-culture, or interactive art projects
